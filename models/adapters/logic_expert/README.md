# Logic Expert Adapter

Domain-specific LoRA adapter for logical reasoning tasks.

## Capabilities

- Formal proofs and theorem proving
- Deductive and inductive reasoning
- Argument analysis and validation
- Truth table construction
- Syllogism evaluation
- Logical fallacy detection

## Training Data

Fine-tuned on:
- Mathematical proof datasets
- Logic textbook exercises
- Formal reasoning benchmarks
- Philosophy argument analysis

## Trigger Keywords

High confidence triggers:
- "prove", "proof", "theorem", "deduce", "infer"
- "therefore", "hence", "implies", "equivalent"
- "valid", "invalid", "premise", "conclusion"
- Logic symbols: quantifiers, connectives
