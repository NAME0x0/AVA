# Coding Expert Adapter

Domain-specific LoRA adapter for software development tasks.

## Capabilities

- Code completion and generation
- Bug detection and debugging
- Code explanation and documentation
- API usage and integration
- Algorithm implementation
- Multi-language support (Python, JavaScript, Rust, Go, etc.)

## Training Data

Fine-tuned on:
- Code completion datasets
- Stack Overflow Q&A pairs
- GitHub code reviews
- Documentation examples

## Trigger Keywords

High confidence triggers:
- "code", "function", "class", "debug", "error", "bug"
- Language names: "python", "javascript", "rust", etc.
- "git", "api", "database", "compile", "test"
