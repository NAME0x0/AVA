# Corpora

Place local language, math, science, coding, and tool-supervision training data here.

- `raw/` is for downloaded or hand-curated sources.
- `processed/` is for cleaned text ready for tokenization.
- `tool_sft/` is the compact supervised packet for early calculator-tool behavior.

These directories are ignored by git, except for the tiny tracked smoke and tool packets used for debugging and session reproducibility.
